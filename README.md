📊 Day 3 – Office Automation Simulation

🔗 Live Demo: https://officeautomationsimulation.netlify.app/

🚀 Project Overview

This project is a simple office automation simulation that demonstrates exporting course/student data to an Excel file using XLSX.js. It showcases how JavaScript can be used to generate a downloadable Excel file to improve productivity and simulate real-world office tasks.

🛠️ Technologies Used

HTML5

CSS3

JavaScript

XLSX.js (SheetJS)

🎯 Objective

To create a system that:

Stores course/student data in JavaScript

Provides an Export to Excel button

Generates and downloads an Excel file containing the data

Offers a clean and simple user interface

📋 Features

✅ One-click Excel export
✅ Downloadable .xlsx file with structured data
✅ Easy to understand implementation
✅ Designed for productivity simulation
✅ Works entirely in the browser — no backend required

🧠 How It Works

Define data in an array (courses/students) inside JS.

When the user clicks Export to Excel:

XLSX.js converts the data into a worksheet.

A workbook is created with that sheet.

XLSX.js triggers the file download in the browser.
